Privacy Policy

Subhadra Report Assistant for Bargarh

Last Updated: February 13, 2026

1. OVERVIEW

This Chrome extension ("Subhadra Report Assistant for Bargarh") is designed for internal use by authorized district officials to automate the collection and synchronization of Subhadra Yojana scheme data. This extension does not collect, store, or transmit any personal user data.

Key Privacy Commitment: We do NOT collect any user data. Your privacy and security are our top priorities.

2. DATA COLLECTION

We do NOT collect any user data.

The extension does not:

  • Collect personally identifiable information
  • Track user behavior or browsing history
  • Store user credentials or passwords
  • Use analytics or telemetry services
  • Send data to any third-party servers
  • Transmit data to the extension developer

3. DATA HANDLING

How the extension works:

Authentication Token Reading

The extension reads the authentication token from the Subhadra portal (subhadra.odisha.gov.in) that you are already logged into. This token is used temporarily to make API requests and is never stored or transmitted elsewhere.

Data Fetching

The extension fetches Field Verification and eKYC data from official government APIs (subhadra-api-dept.odisha.gov.in) using your authenticated session.

Data Writing

Fetched data is written directly to your designated Google Sheet using a service account. No data is sent to any developer-owned server.

Local Processing

All data processing happens locally in your browser. No information leaves your device except to write to your Google Sheet.

4. PERMISSIONS USAGE

The extension requires the following permissions:

activeTab

  • Purpose: To read the session authentication token from the currently open Subhadra portal tab
  • Usage: Only activated when you click the extension icon while on subhadra.odisha.gov.in

scripting

  • Purpose: To access browser LocalStorage to retrieve your login token
  • Usage: Used only to read the authentication token from the Subhadra portal

downloads

  • Purpose: To export reports as Excel (.xlsx) files
  • Usage: Activated only when you click the "Download Excel" button

cookies

  • Purpose: To verify if you are currently logged into the Subhadra portal
  • Usage: Used to check login status before attempting data fetch

5. HOST PERMISSIONS

subhadra.odisha.gov.in
  • Official Subhadra Yojana portal
  • Required to read authentication tokens
subhadra-api-dept.odisha.gov.in
  • Official API endpoint for fetching dashboard data
  • Required to retrieve Field Verification and eKYC statistics
sheets.googleapis.com
  • Google Sheets API endpoint
  • Required to write data to your tracking spreadsheet
oauth2.googleapis.com
  • Google OAuth2 endpoint
  • Required to authenticate service account for Google Sheets access

6. THIRD-PARTY SERVICES

Google Sheets API

  • The extension uses Google Sheets API to write data to your spreadsheet
  • This interaction is governed by Google's Privacy Policy: https://policies.google.com/privacy
  • The extension uses a service account with access only to the designated tracking spreadsheet

Government APIs

  • The extension accesses official Subhadra Yojana APIs operated by the Government of Odisha
  • All data accessed is official government data that you already have permission to view

7. DATA SECURITY

  • All API requests use HTTPS encryption
  • Authentication tokens are used temporarily and never stored permanently
  • Service account credentials are embedded securely within the extension
  • No user data is transmitted to external servers

8. DATA RETENTION

The extension does not retain any user data. All operations are performed in real-time and no information is stored locally or remotely by the extension.

9. CHILDREN'S PRIVACY

This extension is designed for use by government officials and is not intended for use by children under the age of 13.
